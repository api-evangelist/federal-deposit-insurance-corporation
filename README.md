# Federal Deposit Insurance Corporation (federal-deposit-insurance-corporation)

The Federal Deposit Insurance Corporation (FDIC) is an independent agency of the United States government that provides deposit insurance to depositors in US commercial banks and savings institutions. The FDIC also supervises and examines banks for safety and soundness, promotes consumer protection, and publishes the BankFind Suite API for accessing data on FDIC-insured institutions.

## APIs

### FDIC BankFind Suite API

The FDIC BankFind Suite API provides programmatic access to data about FDIC-insured banks and savings institutions, including institution profiles, branch locations, financial summaries, historical records, failures, deposits, and demographic data.

- **Human URL:** https://banks.data.fdic.gov/docs/
- **Base URL:** https://banks.data.fdic.gov/api
